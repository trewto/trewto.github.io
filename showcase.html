Conway's Game of Life With pygame & Python

Conway's Game of Life is a cellular automaton where cells on a grid live, die, or reproduce based on simple rules. My Python implementation simulates these interactions, visualizing how complex patterns emerge from just four basic rules: survival, isolation, overcrowding, and reproduction

AI-powered Flappy Bird using NEAT Algorithm

This project is inspired by the classic Flappy Bird game and was developed as part of a learning exercise using the NEAT (NeuroEvolution of Augmenting Topologies) algorithm. It trains birds to navigate through pipes over multiple generations. Built with Python and Pygame, the neural network adapts and improves its performance, demonstrating key concepts in evolutionary algorithms and neural networks. The project showcases fundamental AI techniques for educational purposes.
